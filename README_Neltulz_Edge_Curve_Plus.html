Key Features

  • Insert single edge loop with flow
  • Insert multiple edge loops with flow
  • Control the tension of the edge loops
  • Control the number of iterations (more iterations = better result)
  • Control the minAngle of the edge loops. Higher numbers can fix problems like, bulging.

Dependencies

Neltulz - Edge Curve Plus requires the EdgeFlow addon for blender to be installed prior to using. Download the latest version here: EdgeFlow Addon for Blender

Special thanks to Benjamin Sauder for creating the EdgeFlow addon for Blender, and implementing some feature requests to make this addon possible!

Installation

  1. Download the Neltulz - Edge Curve Plus zip file anywhere on your computer.
  2. Run Blender.
  3. Go to Edit > Preferences.
  4. In the left sidebar, go to Add-ons.
  5. At the top right, click Install.
  6. Browse for the Neltulz - Edge Curve Plus zip file and click Install Add-on from File
    Alternatively, you may double click the zip file to install it.
  7. Next to Neltulz - Edge Curve Plus, enable the checkbox.
  8. (OPTIONAL) If you are manually saving preferences (instead of automatically), at the lower left of the preferences window, click Save Preferences and close the preferences window.

How to Use

Hotkeys
Note: Hotkey works in both "Object" and "Mesh" mode
Hotkey Result Command Properties
ALT+SHIFT+E Insert Edge Loop(s) with Flow object.neltulz_edge_curve_plus None

Known Issues & Limitations

  • Unable to modify history once edge loops are added.
  • Unable to Repeat Last (workaround: Just use ALT+SHIFT+E hotkey)
