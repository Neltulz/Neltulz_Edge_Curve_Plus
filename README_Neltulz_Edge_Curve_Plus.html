Key Features

  • Insert single edge loop with flow
  • Insert multiple edge loops with flow
  • Control the tension of the edge loops
  • Control the number of iterations (more iterations = better result)
  • Control the minAngle of the edge loops. Higher numbers can fix problems like, bulging.

Dependencies

Neltulz - Edge Curve requires the EdgeFlow addon for blender to be installed prior to using. Download the latest version here: EdgeFlow Addon for Blender

Special thanks to Benjamin Sauder for creating the EdgeFlow addon for Blender, and implementing some feature requests to make this addon possible!

Installation

  1. Download the Neltulz - Edge Curve zip file anywhere on your computer.
  2. Run Blender.
  3. Go to Edit > Preferences.
  4. In the left sidebar, go to Add-ons.
  5. At the top right, click Install.
  6. Browse for the Neltulz - Edge Curve zip file and click Install Add-on from File
    Alternatively, you may double click the zip file to install it.
  7. Next to Neltulz - Edge Curve, enable the checkbox.
  8. (OPTIONAL) If you are manually saving preferences (instead of automatically), at the lower left of the preferences window, click Save Preferences and close the preferences window.

How to Use

  1. Select an object in the 3d viewport
  2. Switch to Edit Mode
  3. Select one or more Edge(s)
  4. Press CTRL+SHIFT+E to insert an edge loop with flow
  5. Alternatively, you may use the Edge Curve+ Sidebar panel to insert edges with flow
  6. Configure the options to your liking!
Hotkeys
Note: Hotkey works in "Edit Mode" only.
Hotkey Result Command Properties
ALT+SHIFT+E Insert Edge Loop(s) with Flow ntzedgcrv.insertedges numSegments = 1
useEdgeFlow = True
tension = 180
numIterations = 4
minAngle = 0
